Hello from the unofficial "Ask BSI Bitesize Immunology"

This is just a fun little test project to understand a bit more about how you can make custom applications to answer questions relating to specific content using Langchain, ChromaDB and OpenAI.
It uses text content from the British Society for Immunology's Bitesize Immunology website, which is a great resource for immunology students.
You can ask a question relating to basic immunology, and if the answer is in the website, you'll get a summary and a link to the relevant page in BSI Bitesize Immunology.
